Introduction and purpose

In the last post we scraped new cancer incidence and population data from the South Carolina government website, merged them together, and ran a model to assess relative cancer rates by county. We then visualized the posterior distributions briefly using the tidybayes package. Here we dig deeper into model diagnostics using tidybayes and also introduce the loo package (stands for leave-one-out).

Setup and Data

We no longer need the rvest and xml2 packages, so we load the same packages as last time, with the addition of loo.

library(tidyverse)
library(viridis)
library(stringr)
library(rstanarm)
library(tidybayes)
library(loo)

options(mc.cores = 4)
